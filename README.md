# modified eCommerce Shopsoftware

A proven open-source eCommerce system based on PHP & MySQL - actively maintained and continuously modernized by the community.

## What is modified?

**modified** is a mature and battle-tested eCommerce shop system used in real-world online stores for many years.

The project focuses on:

- Stability and reliability in production environments
- High extensibility through modules and custom development
- Long-term maintainability
- Practical solutions for real merchants

modified combines classic shop system strengths with continuous technical improvements driven by the community.

## Key Features

- 🛒 Complete eCommerce shop system
- ⚙️ Modular architecture
- 💳 Payment & shipping integrations
- 🌍 Multi-language support
- 🔧 Developer-friendly customization options
- 📦 Large ecosystem of extensions and modules

## Technology Stack

- PHP
- MySQL / MariaDB
- JavaScript
- HTML / CSS
